# Projeto: Site de Impressão 3D

Este repositório contém o desenvolvimento de um site voltado à apresentação de produtos produzidos por impressão 3D. O objetivo principal é fornecer uma vitrine digital organizada e funcional, permitindo a visualização do catálogo e servindo como base para futuras expansões do projeto.

O projeto está sendo desenvolvido com foco em boas práticas de estruturação, organização de código e responsividade, visando uma experiência consistente em diferentes dispositivos.

---

## Visão Geral

A aplicação consiste em uma interface web estática que apresenta produtos de impressão 3D de forma visual e acessível. O sistema foi projetado para ser simples, eficiente e escalável, permitindo a adição de novas funcionalidades conforme a evolução do projeto.

---

## Funcionalidades Atuais

* Exibição de catálogo de produtos
* Organização visual por seções
* Layout responsivo para diferentes tamanhos de tela
* Estrutura modular de estilos (CSS separado por componentes)
* Implementação de efeitos visuais com JavaScript
* Estrutura preparada para expansão futura

---

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

## Execução do Projeto

Para executar o projeto localmente:

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse o diretório do projeto:

```
cd projeto-impressao-3d
```

3. Abra o arquivo `index.html` em um navegador de sua preferência.

Não há necessidade de instalação de dependências, pois trata-se de um projeto estático.
